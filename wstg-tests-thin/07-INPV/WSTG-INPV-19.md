## Brief Summary

This section describes how a tester can check if it is possible to enter code as input on a web page and have it executed by the web server. In [Code Injection](https://owasp.org/www-community/attacks/Code_Injection) testing, a tester submits input that is processed by the web server as dynamic code or as an included file.
