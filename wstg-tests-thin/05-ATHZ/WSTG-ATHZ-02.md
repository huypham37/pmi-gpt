## Brief Summary

This kind of test focuses on verifying how the authorization schema has been implemented for each role or privilege to get access to reserved functions and resources. For every specific role the tester holds during the assessment and for every function and request that the application executes during the post-authentication phase, it is necessary to verify: - Is it possible to access that resource
